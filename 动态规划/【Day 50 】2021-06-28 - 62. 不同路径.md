## 题目地址：

https://leetcode-cn.com/problems/unique-paths/submissions/



## 思路

动态规划



## 代码

JavaScript

```javascript
/**
 * @param {number} m
 * @param {number} n
 * @return {number}
 */
var uniquePaths = function(m, n) {
    const dp = Array(n).fill(1);

    for (let i = 1; i < m; i++) {
        for(let j = 1; j < n; j++) {
            dp[j] = dp[j] + dp[j - 1]
        }
    }
    return dp[n - 1]
};
```



### 复杂度

- 时间复杂度：O(M * N)
- 空间复杂度：O(N)
